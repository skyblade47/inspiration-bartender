# 灵感调酒师 - HTML 演示

> 通过浏览器直接测试 LLM 功能，无需安装任何依赖

## 🎯 功能概述

这个 HTML 演示可以完整测试灵感调酒师的 LLM 相关功能：

- ✅ **LLM 配置** - 支持 OpenAI、Anthropic、Ollama
- ✅ **调酒对话** - 测试 AI 追问功能
- ✅ **灵感评分** - 测试完整度评估
- ✅ **配方生成** - 测试碰撞配方建议
- ✅ **头脑风暴** - 测试创意发散

## 🚀 快速开始

### 1. 打开演示

直接用浏览器打开 `index.html` 文件：

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

或者双击 `index.html` 文件。

### 2. 配置 LLM

在左侧面板配置你的 LLM：

**OpenAI (推荐)**
1. 选择 Provider: `OpenAI`
2. 输入 API Key: `sk-...`
3. 端点留空（使用默认）
4. 选择模型: `GPT-4` 或 `GPT-3.5`

**Anthropic (Claude)**
1. 选择 Provider: `Anthropic`
2. 输入 API Key
3. 模型: `claude-3-opus` 或 `claude-3-sonnet`

**Ollama (本地)**
1. 选择 Provider: `Ollama`
2. API Key 留空
3. 端点: `http://localhost:11434/v1`
4. 模型: `llama3` 或 `mistral`

### 3. 测试连接

点击 **🔗 测试连接** 按钮，确认 LLM 可以正常通信。

### 4. 开始测试

#### 调酒对话测试
1. 切换到 **🍸 吧台场景** 标签
2. 双击任意灵感进入调酒工作台
3. 或点击左侧 **🧪 测试场景** → **调酒对话测试**

#### 灵感碰撞测试
1. 在吧台场景长按选择 2-3 个灵感
2. 点击 **🍸 调制特调** 进入碰撞界面
3. 点击 **🍸 开始调制** 播放动画
4. 查看生成的配方卡片

## 📋 测试场景

### 1. 调酒对话测试 🍸
测试调酒师通过对话引导完善灵感：
- AI 生成追问问题
- 用户输入回答
- 记录对话历史
- 更新灵感完整度

### 2. 灵感评分测试 📊
测试多维度完整度评估：
- 清晰度 (30%)

测试灵感碰撞后的配方建议：
- 多个灵感混合
- AI 生成创意方向
- 评分和关键词提取

### 4. 头脑风暴测试 💡
测试创意发散功能：
- 基于灵感生成关联卡片
- 多维度联想扩展

## 🎨 界面功能

### 左侧面板
- **LLM 配置** - 配置 API 提供商、密钥、端点、模型
- **测试场景** - 快速运行各类测试
- **请求日志** - 查看 API 调用详情

### 主界面
- **🍸 吧台场景** - 展示灵感杯子，支持多选
- **🤔 调酒工作台** - 对话式完善灵感
- **💥 灵感碰撞** - 混合灵感生成配方

## 🔧 技术细节

### LLM 调用
- 支持 OpenAI ChatGPT API
- 支持 Anthropic Claude API
- 支持 Ollama 本地模型
- 支持自定义 API 端点

### 数据流
```
用户输入 → HTML 演示 → LLM API → 响应展示
```

### 本地存储
- LLM 配置保存到浏览器 localStorage
- 无需重复输入配置

## ⚠️ 注意事项

1. **API 费用** - 使用 OpenAI/Anthropic API 会产生费用
2. **网络延迟** - 取决于 API 服务器响应速度
3. **CORS 问题** - 部分 API 可能需要代理服务器
4. **本地模型** - Ollama 需要本地运行
