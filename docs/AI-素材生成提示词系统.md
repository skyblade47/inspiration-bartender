## 一、设计理念

### 1.1 核心目标

生成一套**统一风格、丰富多样**的视觉素材，避免：
- ❌ 各类型素材风格不统一
- ❌ 手绘风格与摄影风格混杂
- ❌ 过于简陋的图标感

### 1.2 视觉风格定位

**风格关键词**：
```
深夜酒吧氛围 · 温暖琥珀灯光 · 精致玻璃质感 · 摄影作品 · 高细节 · 8K分辨率
```

**色彩基调**：
- 主色：琥珀色 (#D4A017)、深棕色 (#3d2e24)
- 辅色：暖橙色 (#FF8C42)、酒红色 (#8B0000)
- 点缀：金色 (#FFD700)、银白色 (#C0C0C0)

**光影效果**：
- 暖黄色顶光从上方照射
- 玻璃表面的光折射
- 液体的半透明质感
- 背景的虚化效果

## 六、技术实现

### 6.1 提示词管理器

```typescript
class PromptGenerator {
  private basePrompt = '深夜酒吧场景，...'
  
  generate(glassType: GlassType, completion: number): string {
    const cupPrompt = this.getCupPrompt(glassType)
    const liquidPrompt = this.getLiquidPrompt(glassType)
    const statusPrompt = this.getStatusPrompt(completion)
    const decorationPrompt = this.randomDecoration()
    
    return [
      this.basePrompt,
      cupPrompt,
      liquidPrompt,
      decorationPrompt,
      statusPrompt,
      this.endingPrompt,
    ].join(', ')
  }
  
  private getCupPrompt(type: GlassType): string {
    const prompts = {
      [GlassType.BRANDY]: '矮胖圆形白兰地杯，...',
      [GlassType.CHAMPAGNE]: '细长香槟杯，...',
      // ...
    }
    return prompts[type]
  }
  
  private getLiquidPrompt(type: GlassType): string {
    const liquids = {
      [GlassType.BRANDY]: '钴蓝色深邃神秘金属光泽',
      [GlassType.CHAMPAGNE]: '淡紫色梦幻浪漫气泡丰富',
      // ...
    }
    return liquids[type]
  }
  
  private randomDecoration(): string {
    const decorations = [
      '柠檬片装饰',
      '薄荷叶点缀',
      '透明冰块漂浮',
      '光线折射效果',
      '',
    ]
    return randomPick(decorations)
  }
}
```

### 6.2 调用示例

```typescript
// 生成 8 种杯子的所有状态
const glassTypes = Object.values(GlassType)
const completions = [0, 25, 50, 75, 100]

for (const type of glassTypes) {
  for (const completion of completions) {
    const prompt = promptGenerator.generate(type, completion)
    await generateImage(prompt, `glass_${type}_${completion}.png`)
  }
}
```

## 七、质量标准

### 7.1 图像质量要求

- **分辨率**：1024×1024 或更高
- **格式**：PNG（透明背景）
- **风格**：统一摄影风格
- **细节**：高细节，高真实感
- **一致性**：同类型素材风格统一

### 7.2 检查清单

- [ ] 所有杯子类型都有 5 种状态
- [ ] 酒液颜色与灵感类型对应
- [ ] 光影效果一致
- [ ] 装饰元素自然
- [ ] 背景虚化适度
