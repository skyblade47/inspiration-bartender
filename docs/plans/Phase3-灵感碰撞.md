### 3.2 碰撞流程

```
选择灵感（2-3个）
    ↓
选择混合类型（分层/混色）
    ↓
开始碰撞动画
    ↓
    ├── 倒入阶段（1.5s）
    ├── 摇晃阶段（2.0s）
    └── 倒出阶段（1.0s）
    ↓
调用LLM生成配方
    ↓
显示3-5个配方卡片
    ↓
用户选择配方或换一批
    ↓
用户确认后创建新灵感
    ↓
记录碰撞历史
```

### 3.3 数据模型

```typescript
interface CollisionRecord {
  id: string;
  sourceInspirationIds: string[];  // 2-3个源灵感
  resultInspirationId?: string;     // 新生成灵感
  
  mixType: MixType;                 // LAYER或BLEND
  mixColors: string[];              // 各灵感颜色
  
  recipes: Recipe[];                // 生成的配方
  selectedRecipe?: Recipe;          // 用户选择的配方
  
  createdAt: number;
}

interface Recipe {
  id: string;
  title: string;                    // 配方标题
  description: string;             // 详细描述
  keywords: string[];              // 关键词
  directions: string[];            // 发展方向
  score: number;                   // 创意评分(0-100)
}
```
